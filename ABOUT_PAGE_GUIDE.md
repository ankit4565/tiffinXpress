## 🎨 Design Features

### Colors Used

- **Primary Orange:** `#ff7a00`
- **Secondary Orange:** `#ff9933`
- **Light Cream:** `#f5f0eb`
- **Dark Text:** `#1f1f1f`
- **Neutral Gray:** `#666`, `#999`

### Typography

- **Headings:** Font-weight 900, bold and impactful
- **Body Text:** Clear, readable, line-height 1.6-1.8
- **Animations:** Smooth CSS transitions

### Interactive Elements

- **Hover Effects:**
  - Cards lift up (translateY)
  - Box shadows increase
  - Colors change to orange accents
  - Scale transforms on stats
- **3D Effects:**
  - Team member cards flip on hover
  - Transform-style preserve-3d
  - Backface-visibility hidden

## 📱 Responsive Breakpoints

- **Desktop (1024px+):** Full 2-column layouts, 3D effects
- **Tablet (768px - 1023px):** 1-column stacked layouts
- **Mobile (480px - 767px):** Single column, optimized spacing
- **Small Mobile (<480px):** Minimal padding, touch-friendly
